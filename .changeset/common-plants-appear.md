New `sl-infotip` component — an info icon button that reveals contextual content in a popover. Designed to be placed in the `infotip` slot of `<sl-label>`, it automatically links its content to the associated form control's native input via `aria-describedby`, so screen readers announce the infotip content when the input receives focus.
