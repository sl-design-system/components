The **SL Design System** is a collection of reusable web components built with [Lit](https://lit.dev/). It provides a consistent, accessible, and themeable set of UI building blocks for Sanoma Learning applications.

## Why a design system?

A shared design system helps teams:

- **Move faster** — Use pre-built, tested components instead of building from scratch
- **Stay consistent** — Every application looks and feels the same
- **Improve accessibility** — Components follow WCAG guidelines out of the box
- **Reduce maintenance** — Fix a bug once, every app benefits

## What's included?

The design system ships several packages:

- **Components** — Lit-based web components (`@sl-design-system/button`, `@sl-design-system/dialog`, etc.)
- **Themes** — CSS design tokens for different brands and applications
- **Tokens** — The raw design token definitions
- **Locales** — Translations for component strings

## Getting help

If you have questions or run into issues, reach out via the [GitHub repository](https://github.com/sl-design-system/components).
