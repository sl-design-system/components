## Button

You can bind actions to buttons using the (click) event.

### Template

```html
<sl-button (click)="action()">Button</sl-button>
```

### Props

```typescript

  action: () => console.log('Button clicked')

```


## Button Bar

A button bar groups multiple buttons together.

### Template

```html
<sl-button-bar>
    <sl-button>Button 1</sl-button>
    <sl-button>Button 2</sl-button>
  </sl-button-bar>
```


## Menu Button

A menu button combines a button with a dropdown menu of options.

### Template

```html
<sl-menu-button>
      <sl-menu-item>Option 1</sl-menu-item>
      <sl-menu-item>Option 2</sl-menu-item>
      <sl-menu-item>Option 3</sl-menu-item>
  </sl-menu-button>
```


## Toggle Button

You can bind actions to buttons using the (sl-toggle) event to respond to toggle state changes.

### Template

```html
<sl-toggle-button (sl-toggle)="action($event)">Toggle me</sl-toggle-button>
```

### Props

```typescript

  action: (event: Event) => console.log('Button toggled, current value:', event)

```


## Toggle Group

A toggle group lets users control related actions with clear visual feedback in a compact UI element.

### Template

```html
<sl-toggle-group>
    <sl-toggle-button>Option 1</sl-toggle-button>
    <sl-toggle-button>Option 2</sl-toggle-button>
    <sl-toggle-button>Option 3</sl-toggle-button>
  </sl-toggle-group>
```
