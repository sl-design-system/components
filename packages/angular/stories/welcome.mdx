# Welcome to the Angular documentation for the Sanoma Learning Design System

This specific part of the documentation is for Angular developers who want to use the Sanoma Learning Design System in their Angular applications.
You will find Angular-specific information, examples, and guidelines here.

The examples in this part don't show the full functionality of the components, for that please refer to the [main documentation website](https://sanomalearning.design). There you will find all the properties, functions and events of each component.
